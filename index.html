Portfolio

FitData

fitdata2016.herokuapp.com

FitData is a cross-platform web application that allows the general public to contribute to health science research through surveys and data collection.

I assisted with the front-end development as I created wireframes and prototypes based on user requirements. I helped enhance the user interface for the data visualization display using jQuery. Lastly, I conducted A/B user-testing using Google Analytics and JavaScript.



Travel Tracker

cogs121-trip.herokuapp.com

Travel Tracker is a desktop web application that allows users to keep track of where they travel, visualize these results using maps and charts, and share these results through social media.

I carried out the entire front-end development process as I used the Google Maps API with jQuery to create an interactive experience with the map. I also used D3.js to create user-friendly visualizations of the travel data. Lastly, I conducted testing sessions with my peers to determine areas of improvement in usability.



Yummit

Sample screens

Yummit is an Android mobile application that allows users to buy and sell homemade food within the community of other users. The sellers can post images and descriptions of the foods for sale and monitor sales whereas the buyers can monitor their placed orders.

I assisted with the front-end development starting from user research all the way to implementation. The front-end was developed using XML for the markup and Java for the user interactions and for linking the client-side with the Firebase database. I used Android Studio to implement Yummit.



Water Tracker

Design brief

Water Tracker is a tablet-based prototype for a water usage monitoring system that allows college students to track their dorm's overall water usage and figure out where in their dorms to reduce water usage.

I helped out with user research to figure out that water is overused in college dorms. Then, I assisted with the ideation process, which led to the water tracking system idea. After ideation, I worked on prototyping until the final prototype met the requirements defined in the user research phase.
