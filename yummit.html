Yummit

Overview

Yummit is a project for a Software Engineering course done in a group of ten members.

Yummit is an Android mobile application that allows users to buy and sell homemade food within the community of other users. The sellers can post images and descriptions of the foods for sale and monitor sales whereas the buyers can monitor their placed orders.

Design Problem

Purchasing homemade food from others is a challenge for college students because they need to be able to find deals that are budget-friendly and nearby.

My Role

I contributed to Yummit through my work on the front-end development of the application. During the planning phase, I assisted with wireframing to help define the information architecture of Yummit.

During the implementation phase of the project, I used XML to create the layout of the user interface and Java to implement the user interaction features.
